Event-Driven Enterprise Simulator

Discrete-event simulation • 2025

Description

A discrete-event simulation of enterprise operations. The model drives entities through queues and resources using an event list (arrivals, service start/finish, departures). It reports throughput, average wait time, and resource utilization. Scenarios are configurable to test different arrival/service distributions and staffing.
